# TextVerified API Research (2026-05-28)

## Overview

TextVerified offers **real US mobile numbers backed by physical SIM cards** — the key differentiator from 5sim/HeroSMS which only have virtual numbers that OpenAI blocks.

- **Website**: https://www.textverified.com
- **Claim**: "Real US mobile numbers backed by physical SIMs – compatible with all platforms"
- **Services**: 900+ supported (Google, PayPal, Uber, Tinder, etc.)
- **API**: REST v2 with official Python SDK (`pip install textverified`)

## API v2 Structure

### Authentication
- **Method**: API Key + Username → Bearer Token (auto-generated by SDK)
- **Headers**: `Authorization: Bearer {token}`, `Accept: application/json`
- **Base URL**: `https://www.textverified.com`

### Key Endpoints

| Action | Method | Path |
|--------|--------|------|
| List services | GET | `/api/pub/v2/services?numberType=mobile&reservationType=verification` |
| Create verification | POST | `/api/pub/v2/verifications` |
| Get verification | GET | `/api/pub/v2/verifications/{id}` |
| List verifications | GET | `/api/pub/v2/verifications` |
| List SMS | GET | `/api/pub/v2/sms?to_number={number}` |
| Account info | GET | `/api/pub/v2/account/me` |
| Cancel verification | POST | `/api/pub/v2/verifications/{id}/cancel` |
| Report verification | POST | `/api/pub/v2/verifications/{id}/report` |
| Price check | POST | `/api/pub/v2/verifications/price-check` |

### Data Types

```python
class NumberType(Enum):
    MOBILE = 'mobile'      # Real phone numbers
    VOIP = 'voip'          # Virtual numbers
    LANDLINE = 'landline'  # Fixed lines

class ReservationCapability(Enum):
    SMS = 'sms'
    VOICE = 'voice'
    SMS_AND_VOICE_COMBO = 'smsAndVoiceCombo'

class ReservationType(Enum):
    RENEWABLE = 'renewable'
    NONRENEWABLE = 'nonrenewable'
    VERIFICATION = 'verification'  # One-time use

class Service:
    service_name: str       # e.g. "openai", "google"
    capability: ReservationCapability

class NewVerificationRequest:
    service_name: str
    capability: ReservationCapability
    area_code_select_option: Optional[List[str]] = None
    carrier_select_option: Optional[List[str]] = None
    service_not_listed_name: Optional[str] = None
    max_price: Optional[float] = None

class VerificationExpanded:
    number: str             # The phone number
    id: str                 # Verification ID
    service_name: str
    state: ReservationState
    total_cost: float
    created_at: datetime
    ends_at: datetime

class PricingSnapshot:
    service_name: str
    price: float
```

## SDK Usage (Python)

```python
import textverified

# Auth
textverified.configure(api_key="...", api_username="...")

# Balance
from textverified import account as tv_account
info = tv_account.me()
print(info.current_balance)

# List services (check if OpenAI exists)
from textverified import services as tv_services
from textverified.data import NumberType, ReservationType
svc_list = tv_services.list(NumberType.MOBILE, ReservationType.VERIFICATION)
for svc in svc_list:
    print(f"{svc.service_name}: {svc.capability}")

# Create verification (get number)
from textverified import verifications as tv_verifications
from textverified.data import ReservationCapability, NumberType
verification = tv_verifications.create(
    service_name="openai",  # or "chatgpt" or use service_not_listed_name
    capability=ReservationCapability.SMS,
    max_price=5.0,
)
print(f"Number: {verification.number}, ID: {verification.id}")

# Poll for SMS
from textverified import sms as tv_sms
messages = tv_sms.list(to_number=verification.number)
for msg in messages:
    print(f"SMS: {msg.text}")

# Cancel
tv_verifications.cancel(verification.id)
```

## Integration Plan

Implement `TextVerifiedProvider(BaseSmsProvider)` in `core/base_sms.py`:

- Constructor: `api_key`, `api_username`, `default_service`
- `get_number()` → create verification via `POST /verifications`
- `get_code()` → poll `GET /sms?to_number={num}`
- `cancel()` → `POST /verifications/{id}/cancel`
- `report_success()` → `POST /verifications/{id}/report`
- Register via `providers/sms/textverified.py` with `register_provider("sms", "textverified_api")`
